## ğŸ†˜ Troubleshooting

### Site Won't Load
- Wait 2-5 minutes after enabling Pages
- Check repository is Public
- Verify index.html is in root folder
- Check Settings â†’ Pages for errors

### Images Missing
- Check file paths: `../images/filename.png`
- Verify images uploaded to `images/` folder
- Filenames are case-sensitive
- Clear browser cache

### CSS Not Working
- Check link paths in HTML
- Verify CSS files uploaded
- Clear browser cache
- Open browser console for errors

### Content Not Updating
- Wait 2 minutes for GitHub rebuild
- Hard refresh browser (Ctrl+Shift+R)
- Check if changes were committed
- Verify correct file was edited

**For more help:** See troubleshooting section in DEPLOYMENT-GUIDE.md
