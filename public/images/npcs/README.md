## How to Add Your NPC Images

### Quick Instructions:

1. **Place all your NPC images in THIS folder** (`public/images/npcs/`)

2. **Recommended image specs:**
   - Size: 200x200 pixels (or similar square ratio)
   - Format: PNG or JPG
   - File size: Under 1MB each

3. **Naming convention** (optional but helpful):
   - Use clear names: `barney.png`, `margaret.png`, `buck.png`
   - Or keep original names and assign via admin panel

4. **Upload methods:**

   **Method A - Direct folder upload:**
   - Copy all images into this folder
   - Commit and push to GitHub
   - Use admin panel to assign to NPCs

   **Method B - Via Admin Panel:**
   - Visit `/admin` on your deployed site
   - Click "Edit" on each NPC
   - Upload image directly through the form
